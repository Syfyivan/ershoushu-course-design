# Development History Note

This branch reconstructs a reviewable development history from the available local course deliverables. The project was completed locally before it was organized into a GitHub repository, so the commits are grouped by functional module to make the implementation easier to review.

The branch does not backdate commits. It is intended to show the project structure, implementation layers, and delivery materials in a clear order:

1. Spring Boot baseline
2. Shared backend infrastructure
3. Data model and MyBatis persistence
4. Service layer
5. REST controllers
6. Front-end user pages
7. Admin management console
8. Database schema and demo records
9. Reviewer documentation
10. Midterm presentation script
